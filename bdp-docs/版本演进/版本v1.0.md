1、 AllData总体来讲，是一个基础平台基于Ambari拉起10+种开源组件的启停，第二个是数据平台的产品

2、由于Apache很多开源项目发展节奏较快，目前暂时没有很好的版本切换的能力，所以维护不同组件的版本比较困难

基于这个前提，打算定期升级维护最新的组件版本，并对已进行二次开发的组件进行适配升级到新的版本，保证以前二开的新功能在新组件版本也能使用
